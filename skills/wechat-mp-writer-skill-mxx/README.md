# wechat-mp-writer-skill-mxx

一个面向微信公众号内容生产的写作型 skill，负责从选题到成稿，再到发布前整理。

它和 `wechat-mp-publisher` 的关系是：
- `wechat-mp-writer-skill-mxx`：负责写、改、润色、配图建议、整理发布参数
- `wechat-mp-publisher`：负责真正调用公众号接口创建草稿 / 发布

## 功能

- 热点选题建议
- 文章大纲生成
- 公众号正文撰写
- AI 去味润色
- 配图建议
- 生成适合公众号的 Markdown
- 对接 `wechat-mp-publisher` 发到草稿箱

## 推荐用法

### 从零开始写

告诉 agent：

- 主题是什么
- 想写给谁看
- 目标是什么（涨粉 / 转化 / 记录 / 传播观点）
- 想要什么风格（干货 / 叙事 / 评论 / 教程）

例如：

```text
帮我写一篇微信公众号文章，主题是 AI 编程，受众是独立开发者，风格偏实战干货，最后给我一个适合公众号的版本。
```

### 已有草稿，帮我润色

```text
把这篇文章改得更像真人写的，少一点 AI 味，适合发公众号。
```

### 写完直接准备发布

```text
把文章整理成微信公众号可发布格式，并准备交给 wechat-mp-publisher 创建草稿。
```

## 去 AI 味策略

这个 skill 的润色重点不是“花哨”，而是“真实”：

- 少模板化结构
- 少官话套话
- 多具体观察
- 多一点口语感
- 允许节奏有起伏
- 保留观点锋利度

## 发布协作方式

如果你同时安装了 `wechat-mp-publisher`：

1. 先让本 skill 生成文章定稿
2. 再让它整理标题 / 作者 / 摘要 / 正文
3. 最后交给 `wechat-mp-publisher` 创建草稿或发布

## 适合谁

- 想持续写公众号但懒得从零搭结构的人
- 有内容但总觉得“像 AI 写的”的人
- 想把内容生产和公众号发布串起来的人
