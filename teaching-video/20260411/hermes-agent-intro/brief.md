# Hermes Agent 入门视频 Brief

- 视频主题：Hermes Agent 入门
- 内容要求：面向已经用过 Claude Code、Codex、OpenClaw 或本地 agent 工具的技术用户，解释 Hermes Agent 解决什么问题、核心能力、国内环境下如何开始试用。
- 风格要求：简洁科技风
- 时间要求：默认 3 分钟，目标 170 到 190 秒
- 语言：中文旁白，屏幕文字使用短中文关键词
- 输出目标：有声音的教学视频，最终 mp4 需要包含音轨

## 学习目标

看完后，观众应该能回答：

1. Hermes Agent 和普通一次性 CLI agent 的差别是什么。
2. 为什么“长期记忆、skills、自我改进、跨平台入口、远程运行”是它的核心价值。
3. 国内技术团队应该如何低风险开始试用。
4. 什么场景适合先试，什么场景应该先观望。

## 内容边界

- 不做完整安装教程，只给 2 小时试用路线。
- 不把海外 IM 或 Google 服务作为默认主路径。
- 不夸大生产可用性，必须提示权限、记忆和远程执行的治理风险。
