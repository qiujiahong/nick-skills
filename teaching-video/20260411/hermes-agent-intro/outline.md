# Hermes Agent 入门视频 Outline

1. 开场：把问题说清楚
   - 普通 agent 能做任务，但长期使用时会反复解释背景。
   - Hermes Agent 的切入点是把 agent 变成长期同事。

2. 它适合谁
   - 已经在用 Claude Code、Codex、OpenClaw 或本地 agent 的用户。
   - 重复工作多、想沉淀 skill、希望多入口和远程长任务的人。

3. 核心能力一：记忆和 skills
   - 记住偏好、项目流程、常见坑。
   - 把做过的复杂任务沉淀成 skill，降低下次启动成本。

4. 核心能力二：多入口和远程运行
   - CLI 之外，可以通过 gateway 接消息平台。
   - terminal backend 支持本机、Docker、SSH、VPS 等运行位置。

5. 国内环境怎么理解
   - 优先本机或 VPS，模型用 Kimi、MiniMax、GLM 或内部 OpenAI-compatible endpoint。
   - 入口优先考虑飞书、钉钉、企业微信等已有协作系统。

6. 两小时试用路线
   - 跑通 CLI。
   - 配好模型。
   - 选择低风险重复任务。
   - 连续跑两三次，看能不能沉淀 skill。
   - 再决定是否接 IM 或长期在线。

7. 结尾判断
   - 值得先试，但不建议立刻全量迁移。
   - 先从低风险、高重复的工作流开始。
